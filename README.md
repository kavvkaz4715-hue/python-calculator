# Простой калькулятор на Python
Это мой первый проект на языке Python.Всу очень просто.Программазапрашивает у пользователя два числа и знак,а затем выводит правильный ответ.
Что умеет этот калькулятор?
* + сложение
* - вычитание
* * умножение
* / деление
! Проверяет  ошибки (если вести неверный знак)
как запустить:
1 Установите **Python** на компьютер.
2 Скачайте файл 'calculator.py'.
3 Запустите файл через терминал командой 'python calculator.py'
