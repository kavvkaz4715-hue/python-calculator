# Простой калькулятор на Python
Это мой первый проект на языке Python.Все очень просто.Программа запрашивает у пользователя два числа и знак,а затем выводит правильный ответ.
Что умеет этот калькулятор?
* + сложение
* - вычитание
* * умножение
* / деление
! Проверяет  ошибки (если ввести неверный знак)
как запустить:
1 Установите **Python** на компьютер.
2 Скачайте файл 'calculator.py'.
3 Запустите файл через терминал командой 'python calculator.py'
